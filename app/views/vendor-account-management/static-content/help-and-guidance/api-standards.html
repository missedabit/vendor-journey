Help and guidance

API standards

You will have received an email on 4th October regarding the API Standards (also provided below). APIs let applications talk to each other and share data, they are essential to our technical process.

As you are an eligible Vendor, we will also be hosting a live Q&A session (between 18th October – 5th November) for any additional questions you might like to raise regarding the API build. A separate email invite will be sent to you on this shortly.

To be able to test API integration, API stubs will be circulated to you on w/c 18th October on the HtG:D Vendor Portal.

About

Help to grow will be presenting a number of APIs for Vendors to engage with. Our APIs will align to Government Digital Service practices. Our APIs will be partner based Web APIs using REST specification.

Use

Our APIs will be available to allow Vendors to interact with our services and will be presented via an api.gov.uk address in due course.

Access

Connecting to the APIs will require you as an organisation to be registered within the Vendor portal and to have provided us with an IP address(es) to allow your systems to connect to us. We will whitelist the IP address and provide an API key for access.

Making requests

When receiving your key, we will also provide our API documentation allowing you to make calls to the system. All access will be monitored for the duration of each session.

For further information, please see the Government Digital Service guidance on API Technical and Data Standards here: https://www.gov.uk/guidance/gds-api-technical-and-data-standards.

If you have any technical questions that are specific to the API or grant token redemption process, please contact: vendors.helptogrow@beis.co.uk
